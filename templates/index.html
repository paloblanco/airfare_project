Accessing the Titanic Passenger API

The following endpoints are available to access data:

  • /api/v1.0/names

    Access a list of all passenger names

  • /api/v1.0/passengers

    Return a list of passenger data including the name, age, and sex of each passenger

  • /api/v1.0/passengerbyname/<name>

    Search for passengers by name. For example:

    /api/v1.0/passengerbyname/Behr,%20Mr%20Karl%20Howell
